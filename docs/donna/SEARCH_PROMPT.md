# Prompt de Investigación Profunda: Proyecto "Donna"

Copia y pega este prompt en tu herramienta de IA favorita (ChatGPT o4, Claude 3.5 Sonnet, Gemini Advanced) para generar una investigación exhaustiva y "bestial" sobre cómo llevar a Donna al siguiente nivel.

---

**PROMPT PARA IA:**

```text
Actúa como un CTO experto en Automatización de Procesos con IA, Arquitectura de Software y Estrategia de Negocios Digitales. Tengo un proyecto llamado "Donna": una Agente de Operaciones basada en IA integrada en mi CRM (Next.js + Supabase).

**CONTEXTO DEL PROYECTO:**
- **Objetivo:** Donna no es un simple chatbot. Es una "Gerente de Operaciones" proactiva. Su misión es liberar al CEO/Closer de tareas repetitivas y carga mental.
- **Stack Actual:** CRM propio (Next.js), Base de datos Supabase.
- **Stack de Automatización Planeado:** VPS con n8n (orquestador), Evolution API (para WhatsApp), y otros agentes vía API.
- **Estado:** Ya tenemos un módulo de "Discovery" que investiga leads con IA (Gemini+Tavily). Ahora queremos expandir a Donna para manejar comunicaciones (WhatsApp/Redes) y agenda.

**TU MISIÓN:**
Realiza una investigación profunda y proponme una estrategia "bestial" para construir a Donna. Cubre los siguientes puntos con detalle técnico y visionario:

1.  **Arquitectura de Agentes Autónomos:** ¿Cómo estructurarías a Donna para que no sea un solo prompt gigante, sino un sistema de mini-agentes (Investigador, Redactor, Planner)? ¿Qué frameworks o patrones recomendarías usar dentro de n8n o Python (ej: LangChain, CrewAI) que se integren bien con mi VPS?

2.  **Humanización Extrema en WhatsApp:** ¿Cómo logramos que Donna responda en WhatsApp (vía Evolution API) sintiéndose 100% humana, con "latencia simulada", "escribiendo...", y manejo de audios (Speech-to-Text y Text-to-Speech ultra realista)? Recomienda herramientas específicas para TTS en español latino.

3.  **Community Management Autónomo:** Quiero que Donna maneje redes. ¿Cómo automatizarías el ciclo: "Detectar tendencia/noticia -> Crear Idea -> Generar Copy+Imagen -> Pedir Aprobación al Humano -> Publicar"? Detalla el flujo en n8n.

4.  **Psicología de Ventas:** ¿Cómo inyectar metodologías de venta (ej: Sandler, Spin Selling) en el contexto de la IA para que al chatear con un lead no solo responda dudas, sino que *califique* y *cierre* citas?

5.  **Benchmarking:** ¿Qué están haciendo los mejores SAAS de automatización en 2024/2025 que debería copiar o mejorar?

Entrégame la respuesta estructurada como un "Whitepaper Técnico y Estratégico" listo para implementar.
```
